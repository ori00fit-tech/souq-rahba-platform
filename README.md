# سوق رحبة | Souq Rahba Platform 🇲🇦

منصة تسوق إلكتروني شاملة موجهة للمغرب العربي، تربط البائعين بالمشترين عبر واجهة عصرية تدعم اللغة العربية بالكامل.

## 📢 نظرة عامة

**سوق رحبة** هي منصة Marketplace متكاملة تم تطويرها باستخدام أحدث تقنيات الويب الحديثة. تتيح للبائعين في المغرب عرض منتجاتهم وخدماتهم، وتوفر للمشترين تجربة تسوق آمنة وسريعة.

> ✅ **الحالة:** المنصة نشطة ومنشورة على [rahba.site](https://rahba.site) 🚀

## 🚀 البدء السريع

### المتطلبات
- Node.js 18+
- pnpm (يوصى به)
- حساب Cloudflare

### الخطوات
```bash
# 1. تثبيت المكتبات
git clone https://github.com/ori00fit-tech/souq-rahba-platform.git
cd souq-rahba-platform
pnpm install

# 2. إعداد قاعدة البيانات
# قم بتحديث ملفات .env الخاصة بـ Cloudflare D1

# 3. تشغيل التطوير
pnpm --filter storefront dev   # لتطبيق الواجهة
pnpm --filter api dev          # لخدمة API

# 4. البناء للنشر
pnpm build
```
